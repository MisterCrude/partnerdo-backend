# Backend-end _partnerdo.pl_

### Run project

#### _Without docker_

- `$ make activate`
- `$ make run`

#### _With docker_

- `$ make up`
- `$ make login`
- `$ make run` (no need to activate pipenv because docker plays role of scoped environment)
